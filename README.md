# casper-cljs

A Leiningen template for [CasperJS][1] projects using [Ghost][2] (ClojureScript CasperJS wrapper).

## Usage

```bash
    lein new casper-cljs myproject
```

[1]: http://casperjs.org
[2]: https://github.com/pandeiro/ghost
